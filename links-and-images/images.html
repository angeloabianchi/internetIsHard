Images

Images are included in web pages with the <"img/>" tag and its src attribute, which points to the image file you want to display.


This page covers common image formats, but you may also be looking for links and useful extras.

JPGs

JPG images are good for photos.

GIFs

GIFs are good for animations.

PNGs

PNGs are good for diagrams and icons.

SVGs

SVGs is a vector-based graphics format, meaning it can scale up or down to any dimension without loss of quality. Use them wherever you can.

We can use the <"img"/> element's width and height attributes to change the size of the image, but it's usually better to set image dimensions with CSS. This attribute were used in JPG example.

Text Alternatives

Adding alt attributes to your <"img"/> elements is a best practice. It defines a "text alternative" to the image being displayed. This attribute were used at GIF example.
