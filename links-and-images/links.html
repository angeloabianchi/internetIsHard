Links

This example is about links and images.

This particular page is about links! There are three kinds of links:

  • Absolute Links - are the most retailed way you can refer to a web resource. Starts with "http://" or "https://" followed by the domain name of the website, then the path.
    Absolute links, like to Mozilla Developer Network, which is a very good resource for web developers.
  • Relative Links - point to another file in your website from the vantage point of the file you're editing. The only thing you need to supply is the path.
    Relative links, like to our extras page.
  • Root-Relative Links - are similar to relative links, but instead of being relative to the current pagem they're relative to the "root" of the entire website.
    The only difference between a root-relative link and a relative one is that the former starts with a forward slash (/), that represent the root of your site.

Link Targets

<"a"> elements also accept a target attribute that defines where to display the page when the user clicks the link. We can use the target attribute to ask the browser to open a link in a new window/tab.
